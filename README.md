📦 Instalador automático do Spicetify via Python

Este projeto em Python executa automaticamente a instalação do Spicetify CLI utilizando o PowerShell, simulando a confirmação do usuário (y) para concluir o processo sem interação manual.

🚀 Como funciona

O script utiliza o módulo nativo subprocess do Python para:

Executar um comando PowerShell

Baixar e rodar o script oficial de instalação do Spicetify

Enviar automaticamente a confirmação (y)

Capturar possíveis saídas e erros do processo

⚙️ Requisitos

Windows com PowerShell instalado

Conexão com a internet

📥 O que é instalado

Este script instala automaticamente o:

Spicetify CLI (ferramenta para customização do Spotify)

⚠️ Avisos importantes

Este script executa código remoto diretamente do GitHub. Use com cautela.

Sempre revise o conteúdo do script remoto antes de executá-lo.

Pode ser necessário executar o terminal como administrador dependendo das permissões do sistema.
